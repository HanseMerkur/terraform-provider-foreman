foreman_operatingsystem¶

Description¶

Foreman representation of an operating system.

Example Usage¶

data "foreman_operatingsystem" "example" {
  title = "CentOS 7.5"
}

Argument Reference¶

The following arguments are supported:

  • title - (schema.TypeString) Title is a Foreman computed property that combines the operating system's name, major, and minor versioning information into a single string.

Attributes Reference¶

The following attributes are exported:

  • architectures - (schema.TypeSet of schema.TypeInt) Identifiers of attached architectures
  • description - (schema.TypeString) Additional operating system information.
  • family - (schema.TypeString) Operating system family. Values include: "AIX", "Altlinux", "Archlinux", "Coreos", "Debian", "Freebsd", "Gentoo", "Junos", "NXOS", "Redhat", "Solaris", "Suse", "Windows".
  • major - (schema.TypeString) Major release version.
  • media - (schema.TypeSet of schema.TypeInt) Identifiers of attached media
  • minor - (schema.TypeString) Minor release version.
  • name - (schema.TypeString) Operating system name.
  • partitiontables - (schema.TypeSet of schema.TypeInt) Identifiers of attached partition tables
  • password_hash - (schema.TypeString) Root password hash function to use. Valid values include: "MD5", "SHA256", "SHA512", "Base64".
  • provisioning_templates - (schema.TypeSet of schema.TypeInt) Identifiers of attached provisioning templates
  • release_name - (schema.TypeString) Code name or release name for the specific operating system version.
  • title - (schema.TypeString) Title is a Foreman computed property that combines the operating system's name, major, and minor versioning information into a single string.
